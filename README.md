# AI Code Review Assistant

An AI-powered code review tool that analyzes source code, detects bugs and code smells, and provides explainable improvement suggestions using Large Language Models.

## Tech Stack
- React
- Node.js (Express)
- LLM APIs (OpenAI / LLaMA)

- The system follows a layered architecture where the frontend handles user interaction, the backend orchestrates static analysis and LLM-based reasoning, and external AI services provide intelligent code review. The design ensures separation of concerns, extensibility, and explainable AI outputs.
